# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly by [opening a private security advisory](https://github.com/lowgrade12/myplugins/security/advisories/new) on this repository.

Please do **not** open a public issue for security vulnerabilities.

## Supported Versions

Only the latest version of each plugin is supported with security updates.

## Security Best Practices

- **Never commit secrets or API keys** to this repository.
- Plugin code runs in the browser context of your Stash instance. Only install plugins you trust.
- Keep your Stash instance updated to the latest version.
